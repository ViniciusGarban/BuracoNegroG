🌌 # Project: Black Holes

This project consists of the development of a website with an astronomical theme, focused on presenting concepts related to black holes in a structured and visual way.

The application integrates both front-end and back-end features, including a user authentication system (login and registration), real-time data validation, and API communication.

## Main Features

- Navigation through informational sections about:
  - Origin of black holes
  - Event horizon
  - Wormholes
- User registration and login system
- Password validation with security requirements
- Language switching (PT-BR / EN)
- Interface designed with focus on visual organization and usability

## Technologies Used

- HTML
- CSS
- JavaScript
- Node.js
- MySQL

## Objective

This project was developed for academic purposes, aiming to apply concepts of web development, API integration, and interactive interface design.

🌌 # Projeto: Buracos Negros

Este projeto consiste no desenvolvimento de um site com temática astronômica, voltado à apresentação de conceitos relacionados a buracos negros de forma visual e estruturada.

A aplicação integra recursos de front-end e back-end, incluindo um sistema de autenticação de usuários (login e cadastro), validação de dados em tempo real e comunicação com API.

## Funcionalidades principais

- Navegação por seções informativas sobre:
  - Origem de buracos negros
  - Horizonte de eventos
  - Buracos de minhoca
- Sistema de cadastro e login de usuários
- Validação de senha com critérios de segurança
- Alternância de idioma (PT-BR / EN)
- Interface com foco em organização visual e usabilidade

## Tecnologias utilizadas

- HTML
- CSS
- JavaScript
- Node.js
- MySQL

## Objetivo

O projeto foi desenvolvido com finalidade acadêmica, com o objetivo de aplicar conceitos de desenvolvimento web, integração com API e construção de interfaces interativas.
